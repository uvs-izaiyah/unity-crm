# Demo Script

## 1. Login + Dashboard
1. Open the app and log in as `founder@commandcenter.test`.
2. Call out the Today’s 5 Moves card and explain the daily 6am generator.
3. Highlight hot leads, content to post, calls today, and weekly revenue metrics.

## 2. Content Pipeline
1. Navigate to the content board.
2. Drag a card from IDEA → RECORDED → POSTED (demo script only).
3. Trigger the automation endpoint for POSTED content to create a repurpose task.

## 3. Deals Pipeline
1. Open the deals kanban board.
2. Move a deal from BOOKED → PAID.
3. Explain that the fulfillment checklist auto-creates when status hits PAID.

## 4. Fulfillment Checklist
1. Open the checklist for the paid deal.
2. Mark items complete to show progress.

## 5. Automation + Tasks
1. Mark a task DONE and point out the follow-up task created.

## 6. Wrap Up
1. Show audit logs entry for each automated action.
2. Review proof pack in `/PROOF`.
